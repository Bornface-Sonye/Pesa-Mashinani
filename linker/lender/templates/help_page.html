Getting Started

Welcome to Pesa Mashinani!
To get started:

  • Register as a user by following the guidelines for your role (Group, Borrower, Bank).
  • Log in using the credentials provided upon registration.
  • Explore the dashboard to access various features and functionalities.

  • Security Tips: Use a strong password and enable two-factor authentication if available to enhance your account security.

System Features

Explore the key features of Pesa Mashinani:

  • Borrower Interface: Register, apply for loans, view loan details, and make repayments.
  • Group Interface: Manage group operations, allocate and disburse loans, update member details, and view financial transactions.
  • Bank Interface: Register, allocate loan amounts, provide interest rates, and list defaulters.
  • Admin Interface: Register and manage groups, borrowers, and lenders. Generate usernames, and handle system-wide operations.

Guidelines for Registration

For Groups

  1. Download the Guarantor Form.
  2. Complete the form and get it signed and stamped by a guarantor.
  3. Submit the signed form to the Pesa Mashinani registration office.
  4. Upon successful registration, you will receive a username.
  5. Use the username to sign up and create your account, then log in and start using the system.

For Banks

  1. Download the Bank Registration Form.
  2. Fill out and sign the form with the necessary stamp.
  3. Submit the form to the Pesa Mashinani registration office.
  4. After registration, you will receive a username.
  5. Create your account using the provided username, then log in to start using the system.

For Borrowers

  1. Download the Borrower Registration Form.
  2. Complete the form with the required personal information.
  3. Submit the form along with any required documents (e.g., identification, proof of group membership) to the Pesa Mashinani registration office.
  4. After successful registration, you will receive a username.
  5. Use the username to sign up and create your account, then log in and start using the system.
